# Through My Lens ????

Welcome to **Through My Lens**, a modern, high-fidelity social media application designed to let users share their perspectives and memories geospatially across a 3D globe and traditional social feeds.

## Features ?
- **Crystal Glass Morphism Design:** Stunning, translucent, premium visuals.
- **Hybrid Navigation:** Switch between a standard 2D layout and a fully interactive 3D WebGL Globe view.
- **Left Sidebar Hub:** Robust, intuitive navigation system.
- **Immersive Post Previews:** Explore memories in a full-screen cinematic modal.
- **Geospatial Discovery:** Drop pins to find new interactions around you.
- **Friends System:** Manage privacy by sharing exclusively with your friends list.

## Technologies Used ??
- React (Vite)
- Firebase (Firestore & Auth)
- Framer Motion
- Globe.gl
- Lucide Icons
- Vanilla CSS (Glass Morphism overrides)

## Getting Started ??
1. Clone the repository
2. Install dependencies: cd web && npm install
3. Run the development server: 
pm run dev
